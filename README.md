# HikarinSDK

## 仿MobTalkerSDK编程风格hikarin-framework

[MobTalkerSDK](https://github.com/Iteranya/MobTalkerSDK) 是旧的 [MinecraftHikarinMod](https://github.com/Iteranya/MinecraftHikarinMod) 剧情脚本制作工具包，其项目结构简单，不使用图形化代码块编程，只要使用 `Python` 脚本。  
此项目尝试移除 hikarin-framework 的图形化界面，只保留核心 `json` 剧情脚本编译器，还原 MobTalkerSDK 的项目结构，为只想使用 `Python` 编写剧情脚本的开发者简化开发环境。
